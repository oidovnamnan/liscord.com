# 🏢 LISCORD — Бизнес Ангилалууд & Тусгай Нөхцлүүд

> **Зарчим:** Бизнес бүр өөрийн өвөрмөц нөхцөлтэй. Liscord бүртгүүлэх үедээ ангилалаа
> сонговол, тухайн ангилалд тохирсон талбарууд, статус workflow, тайлан автоматаар
> идэвхждэг. Ерөнхий суурь бүтэц нэг боловч, ангилал бүрт **нэмэлт тусгай талбарууд**,
> **өөрийн статус урсгал**, **өвөрмөц функцууд** байна.

## 🔷 АНГИЛАЛ A: КАРГО / ИМПОРТ

> **Хэнд зориулсан:** Хятад, Солонгос, Японоос бараа импортлож, Монголд хүргэдэг карго бизнесүүд.
> Жишээ: "Эрээн карго", "Солонгос карго", олон нийтийн карго үйлчилгээ.

### A.1 Тусгай захиалгын талбарууд
| Талбар | Төрөл | Заавал? | Тайлбар |
|--------|--------|---------|---------|
| `trackingNumber` | string | ✅ | Бараа мөшгих дугаар (tracking ID) |
| `senderName` | string | ❌ | Илгээгчийн нэр |
| `originCountry` | select | ✅ | Гарал үүсэл: Хятад / Солонгос / Япон / Бусад |
| `originCity` | string | ❌ | Хот: Эрээн, Гуанжу, Бээжин гэх мэт |
| `weight` | number | ✅ | Жин (кг) |
| `volume` | number | ❌ | Эзэлхүүн (м³) |
| `dimensions` | object | ❌ | Хэмжээ: урт × өргөн × өндөр (см) |
| `packageCount` | number | ✅ | Боодлын тоо |
| `packageType` | select | ❌ | Хайрцаг / Уут / Паллет / Бусад |
| `contentDescription` | text | ✅ | Агуулгын тодорхойлолт |
| `declaredValue` | number | ❌ | Зарласан үнэ (гаалийн) |
| `shippingMethod` | select | ✅ | Авто / Төмөр зам / Агаар / Далай |
| `containerNumber` | string | ❌ | Чингэлгийн дугаар |
| `batchId` | string | ❌ | Ачааны багцын дугаар (нэг рейсийн) |
| `estimatedArrival` | date | ❌ | Хүлээгдэж буй ирэх огноо |
| `actualArrival` | date | ❌ | Бодит ирсэн огноо |
| `pickupLocation` | select | ✅ | Авах байршил / салбар |
| `customsFee` | number | ❌ | Гаалийн татвар |
| `shippingFee` | number | ✅ | Тээврийн төлбөр |
| `insuranceFee` | number | ❌ | Даатгалын төлбөр |
| `storageDeadline` | date | ❌ | Хадгалах хугацааны эцсийн өдөр |
| `isFragile` | boolean | ❌ | Эмзэг бараа эсэх |
| `isOversized` | boolean | ❌ | Том овортой эсэх |
| `photo` | image[] | ❌ | Барааны зураг (хүлээн авахад) |

### A.2 Статус Workflow
```
📥 Захиалга хүлээн авсан
    ↓
🏭 Агуулахад ирсэн (гадаад)
    ↓
🚢 Тээвэрлэж байна
    ↓
🛃 Гаалинд байна
    ↓
📦 Монголд ирсэн (агуулахад)
    ↓
📞 Харилцагчид мэдэгдсэн
    ↓
🤝 Хүлээлгэж өгсөн
    ↓
✅ Дууссан

❌ Цуцалсан (ямар ч үе шатнаас)
🔙 Буцаагдсан
⚠️ Асуудалтай (гааль, гэмтэл гэх мэт)
```

### A.3 Барааны тусгай шинж чанар
- Ангилал: Электроник / Хувцас / Хүнс / Гэр ахуй / Тоног төхөөрөмж / Бусад
- Гаалийн код (HS code)
- Хориглосон/хязгаарлагдсан бараа эсэх

### A.4 Харилцагчийн нэмэлт талбарууд
| Талбар | Тайлбар |
|--------|---------|
| `passportNumber` | Гадаад паспортын дугаар |
| `registrationNumber` | Регистрийн дугаар |
| `preferredPickupLocation` | Байнга авдаг салбар |
| `deliveryAddress` | Хүргэлтийн хаяг |

### A.5 Үнийн тооцоо
- **Жингээр:** ₮X / кг (жишээ: ₮8,000/кг агаараар, ₮3,500/кг авто)
- **Эзлэхүүнээр:** ₮X / м³
- **Хольмог:** MAX(жин, эзлэхүүн) × тариф
- **Нэмэлт:** Гаалийн татвар + Даатгал + Хадгалалт

### A.6 Тусгай функцууд
- 📊 **Ачааны багц удирдлага** — Нэг рейсийн бүх захиалгыг бүлэглэх
- 📱 **Tracking хуудас** — Харилцагч tracking дугаараар шалгах (нийтийн хуудас)
- 💰 **Автомат үнийн тооцоо** — Жин × тариф автомат
- 📋 **Гаалийн мэдүүлэг** — Бараа жагсаалтаас автомат үүсгэх
- 📦 **Агуулахын удирдлага** — Хаана байгаа, хэдэн өдөр хадгалагдсан
- 🔔 **Мэдэгдэл:** Бараа ирлээ → SMS/Push

### A.7 Dashboard KPI
- Өнөөдөр ирсэн ачааны тоо
- Замд яваа ачааны тоо
- Хүлээж аваагүй (авахаар мэдэгдсэн боловч аваагүй)
- Хадгалалтын хугацаа хэтэрсэн
- Тухайн рейсийн нийт жин / дүн
- Гарал үүслийн улсаар нэгтгэл

### B.2 Статус Workflow
```
📥 Захиалга хүлээн авсан
    ↓
✅ Баталсан (үнэ тохирсон)
    ↓
📦 Бэлтгэж байна (агуулахаас цуглуулж байна)
    ↓
📋 Нэхэмжлэл илгээсэн
    ↓
🚚 Хүргэлтэнд гарсан / Авахаар бэлэн
    ↓
🤝 Хүлээлгэж өгсөн
    ↓
💰 Төлбөр хүлээгдэж буй (зээлтэй бол)
    ↓
✅ Бүрэн төлсөн — Дууссан

❌ Цуцалсан
🔙 Буцаалт хийгдсэн (хэсэгчлэн / бүрэн)
```

### B.3 Барааны тусгай шинж чанар
| Талбар | Тайлбар |
|--------|---------|
| `retailPrice` | Жижиглэн үнэ |
| `wholesalePrice` | Бөөний үнэ |
| `tierPrices[]` | Шатлал үнэ: [{min: 10, price: 50000}, {min: 50, price: 45000}] |
| `minOrderQty` | Хамгийн бага захиалгын тоо |

### B.4 Харилцагчийн нэмэлт талбарууд
| Талбар | Тайлбар |
|--------|---------|
| `companyName` | Компанийн нэр |
| `companyRegister` | Компанийн регистр |
| `taxId` | Татвар төлөгчийн дугаар |
| `bankAccount` | Дансны мэдээлэл |
| `creditLimit` | Зээлийн хязгаар |
| `currentBalance` | Одоогийн тооцооны үлдэгдэл |
| `paymentHistory` | Төлбөрийн түүх |
| `priceLevel` | Харилцагчийн үнийн шат |
| `salesRepresentative` | Хариуцсан менежер |

### B.5 Тусгай функцууд
- 💰 **Авлага/өглөгийн удирдлага** — Хэн хэдий төлөх ёстой
- 📊 **Шатлал үнэ** — Тоо ширхэгээс хамааран автомат үнэ
- 🔄 **Давтагддаг захиалга** — Тогтмол захиалга автомат үүсгэх
- 📄 **Нэхэмжлэл** — Автомат нэхэмжлэл PDF үүсгэх
- 📋 **PO (Purchase Order)** — Худалдан авалтын захиалга
- ⚠️ **Зээлийн хяналт** — Хязгаар хэтрэхэд анхааруулах
- 📈 **Харилцагчийн ABC шинжилгээ** — Топ / дунд / бага харилцагчид

### B.6 Dashboard KPI
- Нийт авлага (accounts receivable)
- Хугацаа хэтэрсэн авлага
- Өнөөдрийн борлуулалт
- Топ 10 харилцагч (борлуулалтаар)
- Нөөц дуусч буй бараа
- Давтагддаг захиалгын хуваарь

### Firestore-д хадгалах бүтэц
```javascript
// businesses/{businessId}
{
  name: "Эрээн Карго",
  category: "cargo_import",      // ← Бизнесийн ангилал
  categoryConfig: {
    // Тухайн ангилалд тусгай тохиргоо
    enabledFields: ["trackingNumber", "weight", "originCountry", ...],
    statuses: [
      { id: "received", label: "Хүлээн авсан", icon: "📥" },
      { id: "warehouse_abroad", label: "Агуулахад (гадаад)", icon: "🏭" },
      { id: "in_transit", label: "Тээвэрлэж байна", icon: "🚢" },
      // ...ангилал бүрт өөр
    ],
    customFields: [
      // Бизнес эзэн өөрөө нэмж болох нэмэлт талбарууд
      { id: "custom_1", label: "Тусгай тэмдэглэл", type: "text" }
    ],
    pricingModel: "weight_based",  // cargo: жингээр, wholesale: шатлал гэх мэт
  }
}
```

### Custom Fields систем
Бизнес эзэн тохиргооноосоо **нэмэлт талбар** өөрөө үүсгэж болно:
```javascript
// Тохируулж болох талбарын төрлүүд:
- text        // Текст
- number      // Тоо
- select      // Сонголт (жагсаалтаас)
- multiselect // Олон сонголт
- date        // Огноо
- datetime    // Огноо + Цаг
- boolean     // Тийм/Үгүй
- image       // Зураг
- file        // Файл
- phone       // Утасны дугаар
- currency    // Мөнгөн дүн
```
